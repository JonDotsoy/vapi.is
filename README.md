# vapi.is

Validation of type.

## API Supported
- `is`
- `isA`
- `isAn`
- `isAny`
- `isArray`
- `isFrozen`
- `isFunction`
- `isNumber`
- `isObject`
- `isSealed`
- `isString`

## How to use

Install.

```shell
# With npm
npm install --save vapi.is

# With yarn
yarn add vapi.is
```

Import **vapi.is**.

```javascript
// Import all validations
import * as validations from 'vapi.is'
// A especific validation
import {isString} from 'vapi.is/isString'
// no recommended
import {isString} from 'vapi.is'
```
